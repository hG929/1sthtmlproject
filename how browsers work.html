How does a browser take you to a website?

As you might be wondering, how people are able to key in some letters and numbers and the browsers will take them to a specific page online.

So here is how a browser works

  1. A browsers takes the link (or URL) and send it to another online service called the DNS (Domain Name System)
  2. A DNS takes a URL and transfer it to a IP address, like google.com to 113.245.123.11 (numbers between 1 and 255)
  3. An IP address of a URL is the address of the web server that stores the web content of the web page.
  4. A web server's jobs is to send and receive data to and from the browser
  5. The browser takes data from a web server and build a web page for people to read.
